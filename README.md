PhotoBrowser
============

A simple iOS Instagram photo browser written in Swift.

## Libraries Used

- [Alamofire](https://github.com/Alamofire/Alamofire)
- [SwiftyJSON](https://github.com/SwiftyJSON/SwiftyJSON)
- [FastImageCache](https://github.com/path/FastImageCache)

## Requirements

- Xcode 6.1.1
- iOS 8
